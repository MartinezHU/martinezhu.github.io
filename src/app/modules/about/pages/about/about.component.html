Sobre Mí

¡Hola! Soy Héctor Martínez, un desarrollador web apasionado por crear soluciones innovadoras y funcionales. Mi carrera comenzó con mis estudios en un grado medio de Sistemas Microinformáticos y Redes y un grado superior de Desarrollo de Aplicaciones Web, donde me introduje en el mundo de la programación con Java, Python y tecnologías web modernas. Desde entonces, he trabajado en una variedad de proyectos, tanto personales como profesionales, que abarcan desde aplicaciones web y más recientemente aplicaciones móviles.

En mi última experiencia profesional, desarrollé aplicaciones web utilizando Angular con herramientas como PrimeNG y NgRx. Gestioné bases de datos SQL Server y trabajé con .Net Framework 4.8 para desarrollar backends con autenticación JWT. También tengo experiencia con Java / JSP y Django / Django Rest Framework para desarrollar aplicaciones web o API's REST.

Además, mi interés por la arquitectura de software se refleja en mi comprensión y aplicación de patrones de diseño como CQRS, Patrón Repository, SOLID y Clean Architecture. También tengo un sólido conocimiento en Docker y Docker Compose para la gestión de contenedores y entornos.

Actualmente, estoy trabajando en un proyecto personal desarrollando una red social con Flutter, donde estoy implementando Riverpod para la gestión de estado y enfrentando el desafío de la autenticación utilizando datos biométricos. Me encanta modularizar proyectos y asegurarme de que sean fáciles de entender y mantener.

Mi objetivo es seguir aprendiendo y creciendo en el campo del desarrollo de software, mientras aplico mis conocimientos para resolver problemas reales y contribuir a proyectos innovadores.
